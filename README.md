AI Chatbot (Laravel + Vue + Vuex)
###
This project is a simple AI-powered chatbot built with Laravel and Vue, using Vuex for state management and OpenAI API for AI responses.

**Features**

AI chat using OpenAI Chat Completion API

Function calling support (example: sum_numbers)

Laravel service-based AI architecture

Vue frontend with real-time chat UI

Vuex store for managing chat messages and state

Clean separation between backend (AI logic) and frontend (UI)

**Tech Stack**
Backend: Laravel

Frontend: Vue

State Management: Vuex

AI: OpenAI API (Chat + Function Calling)

**Current Status**

AI chatbot integration completed

Function tool (sum_numbers) implemented

Ready for extending more tools and prompts

**Future Improvements**

Streaming responses

Conversation history storage

More AI tools and structured outputs

Authentication and user-based chats
